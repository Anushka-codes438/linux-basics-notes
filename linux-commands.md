# linux Basic commands 

## File and directory commands
-ls 
--> shows the list of  directory and files.

- pwd
--> prints the current working directory.
  
- cd
--> stands for change directory .  it is used to move from one directory to another in the terminal.
  
- mkdir
--> creates a directory.
  
- touch
--> creates a file.
  
- rm
--> removes a file.
  
- rm -r
--> removes a directory and its contents recursively.
  
- rmdir
--> removes an empty directory.
  
- ls -l
--> shows detailed information about files and directories.
  
  ## File Operations
  
- cp
--> It is used to copy file into another folder or file.
  
- mv
--> It is used to move the file into folder or file.
  
- echo
--> It is used to print the text/output to the terminal.
  
 ## Links 
 - hard link (ln)
--> It is used to create a shorcut of the file of the original file.
   
- soft link(ln -s)
--> It is  used to create  shorcut of the files . if the  original file is  deleted, then soft link becomes broken.
  example:
  ...bash 
  ln file1 file2
  ln -s file1 softlink
