 ## Linux Basics for Cloud Engineers – Part 2
- cut
-> It is used to extracts specific columns or characters from a text or file.

- tee
-> It is used to display command  output on the terminal and simultaneously write it to a file.

- sort
-> It is used to arrange data in the alphabetical order.

  sort -n : numerical
  sort -r : reverse

- diff
-> It is used to show the differnces between two file.

-> - vi editor  
→ It is used to edit text files.

## Login related
 
- ssh (secure shell) 
→ It is used to securely connect to a remote system or server.

Mostly used for:
AWS EC2 login
Cloud servers access
Remote administration

- syntax
ssh username@server_ip

## Disk usage
- du
-> It is used to show the disk usage of files and directories.
-> du -sh  → total size of a directory

-df 
→ Used to display total, used, and available disk space of file systems.
-> df -h   → human readable disk usage

## Process
- top  
→ It is used to display running processes and system resource usage in real time.

- ps  
→ It is used to display information about currently running processes.

- kill
-> It is used to terminate current process.

 - free  
→ It is used to display memory (RAM) usage.

- nohup  
→ It is used to run a command in the background even after the terminal is closed.
