# Linux Introduction 
## What is Linux?
--> Linux is an  open-source operating system.

## why linux is used for cloud?
--> open-source and free.
    - highly secure
    - lightweight and fast
    - easy automation with scripts
    - most cloud servers run on linux

## What is server?
--> A server provides services to clients.

## Types of server
  - web server
  - application server
  - database server 

## What is client?
-->  A client requests a  server to provide information or services.
ex - telephone,laptop,tablet.

## Differnce between Linux and Unix.
Linux - It is open source operating system.
        it is licensed under GPL(General pupose Licence).

Unix - it is a paid operating system.
        ex -  macos
        
## What is Kernel?
--> Kernel is the core part of the operating system.

## what is shell?
--> shell is the terminal or we can say it is communication btween kernel and shell thorugh shell commands.

## what is bootloader?
--> Bootloader is the first step in the operating System boot process.
for ex - GRUB,LILO

## Linux booting process(overview)
- BIOS/UFEI (starts system)
- bootloader loads kernel
- kernel load system service
- user login screen appears
